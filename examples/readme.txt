# MultiRepMacsChIPSeq pipeline examples

These are example pipelines for a variety of situations. 

For details visit:

https://huntsmancancerinstitute.github.io/MultiRepMacsChIPSeq/Examples.html

To build the project and run the first example pipeline script, run the following:

    perl Build.PL
    ./Build
    cd examples
    bash ./run_se.sh
